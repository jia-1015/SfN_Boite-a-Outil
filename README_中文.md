# SfN Tool – Version 6

这一版完成了第6步 `Analyser et présenter`，六步主流程现在已经贯通。

## 完整评价路径

当第3步回答 `Oui` 时：

1. 第4步筛选 impacts；
2. 第5步选择评价层级并填写结果；
3. 第6步自动生成综合报告。

第6步会把 impacts 分成：

- Coûts；
- Impacts négatifs；
- Bénéfices。

综合表显示：

- 选择的方法层级；
- 定性结果；
- 定量结果；
- 货币结果；
- 相对重要性；
- 不确定性。

页面还会显示：

- 已填写 impact 数量；
- 定量结果数量；
- 货币结果数量；
- importance forte 数量；
- impact 相对重要性的横向图。

## 简化评价路径

当第3步回答 `Non` 时，会直接进入第6步的简化报告，可填写：

- 实施成本；
- 预期社会经济效果；
- 为什么采用简化分析；
- 不确定性和注意事项。

## 打印和 PDF

第6步底部有：

`Imprimer / enregistrer en PDF`

点击后会调用浏览器打印功能。可以在打印窗口中选择：

`Enregistrer au format PDF`

打印版本会自动隐藏导航按钮和输入表单。

## 测试建议

### 路径一

- 第3步选择 `Oui`；
- 第4步保留几个 impacts；
- 第5步填写不同类型的结果和 importance；
- 第6步检查三类综合表及图表。

### 路径二

- 返回第3步并选择 `Non`；
- 在第6步填写简化报告；
- 测试打印或保存 PDF。

# Publication sur GitHub Pages

Ce dossier contient le site statique prêt à être publié.

Important : lors de l’envoi sur GitHub, le fichier `index.html` doit être placé
directement à la racine du dépôt, avec les dossiers `css`, `js`, `data` et `images`.

La racine du dépôt doit ressembler à ceci :

index.html
.nojekyll
css/
js/
data/
images/

Ne téléversez pas uniquement le dossier `sfn-tool-public` comme sous-dossier,
sinon GitHub Pages ne trouvera pas correctement la page d’accueil.
